# MGAE-DC

In the "Comparison between different types of models" experiment of the study "A Review of Deep Learning Approaches for Drug Synergy Prediction in Cancer", MGAE-DC is retrained using the DrugComb dataset, and the hyperparameters are readjusted to ensure a fair comparison under consistent experimental conditions:

*batchsize=256
*learning rate=0.0001
*epoch=500 

## Requirements
Python 3.8 or higher  
pandas 1.3.5
numpy 1.21.2
tensorflow 2.4.1
